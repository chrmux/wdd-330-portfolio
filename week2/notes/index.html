Chapter 4: Functions

  • Defining a Function
  • Defining a function literal using a function declaration:
    • function hello() {
        console.log('Hello World!');
    }
  • Defing a function literal using a function expression (This assigns ananonymous function to a variable):
    • const goodbye = function(){
        console.log('Goodbye World!');
    };
  • Defing a function literal using a named function expression
    • const goodbye = function bye(){
        console.log('Goodbye World!');
    };
  • REMEMBER! Every function has a name!
